Range Length

Description

Ensures that the value of the property is within the given range.

Default Message

Value must be between {min} and {max} characters

Template Fields

  • {field} - The property name
  • {min} - The minimum value
  • {max} - The maximum value

Usages

public string AtLeastFiveButNotTen { get; set; }

Property(x => x.AtLeastFiveButNotTen).RangeLength(5, 10);
