Maximum Length

Description

Ensures that the length of a string is less than or equal to the given value.

Default Message

Maximum length not met. Must be less than or equal to {length}.

Template Fields

  • {field} - The property name
  • {length} - The maximum length

Usages

public string NoMoreThanFiveCharacters { get; set; }

Property(x => x.NoMoreThanFiveCharacters).MaximumLength(5);
