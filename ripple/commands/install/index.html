Install

ripple installInstallCommand

ripple install [-v, --version ] [-m, --modeflag Fixed|Float] [-p, --project ] [-f, --force-updates] [-p, --preview] [-s, --stability ] [-l, --solution ] [-c, --cache ] [-a, --all] [-v, --verbose]

Arguments
packageThe NuGet to install

Flags
[-v, --version <version>]Version of the package
[-m, --modeflag Fixed|Float]The update mode of the package
[-p, --project <project>]Project to install to
[-f, --force-updates]Forces updates of transitive updates
[-p, --preview]Only show what would be installed
[-s, --stability <stability>]Override the feed-level stability configuration
[-l, --solution <solution>]override the solution to be cleaned
[-c, --cache <cache>]Override the NuGet cache
[-a, --all]Apply restore to all solutions
[-v, --verbose]Writes all output to the screen
