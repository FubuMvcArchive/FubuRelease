Commands

Clean

ripple cleanCleans out existing package folders and/or project bin & obj directories of existing build products

ripple clean[-m, --mode all|packages|projects] [-l, --solution ] [-c, --cache ] [-a, --all] [-v, --verbose]
Flags
[-m, --mode all|packages|projects]choose what gets cleaned up
[-l, --solution <solution>]override the solution to be cleaned
[-c, --cache <cache>]Override the NuGet cache
[-a, --all]Apply restore to all solutions
[-v, --verbose]Writes all output to the screen
