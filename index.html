  • Web Applications

    Build compositional web applications with FubuMVC. Easily generate html in .Net with the HtmlTags library.


  • Event Based Architecture

    Use FubuTransportation as a low friction service bus for your messaging infrastucture needs.


  • Foundational Tools

    Cleanly compose your application's services and dependencies with StructureMap, the original IoC/DI tool for .Net. Use the FubuCore library for common tasks inexplicably left out of .Net's Base Class Library.


  • Automated Testing

    Write and manage executable specifications in user facing language with the Storyteller tool. Use Serenity as the glue for hosting and exercising FubuMVC and FubuTransportation applications with Storyteller.


  • Build Automation

    The combination of Rake and our own FubuRake gem makes generic build automation and integration with many of the tools on this site a breeze.


  • Dependency Management

    Use ripple for better dependency management with Nuget -- especially for continuation integration scenarios.


  • Console Applications

    Build command line interfaces with the command line parsing tooling in FubuCore.


  • Windows Services

    Quickly host FubuMVC or FubuTransportation applications in windows services using BottleServiceRunner from the Bottles project and the most excellent TopShelf library.


  • Documentation Generation

    Living technical documentation (like this site!) made easy with FubuDocs.


  • Extensibility

    Build extensible packaging capabilities into your own framework or application with Bottles.
