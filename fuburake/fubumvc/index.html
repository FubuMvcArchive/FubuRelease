FubuMVC Applications

To enable Katana hosting for either FubuMVC or FubuTransportation applications, simple use the FubuRake::MvcApp helper like this snippet:

FubuRake::MvcApp.new({:directory => 'src/Music', :name => 'Music'})

Which will give you these rake tasks using music as the task prefix from the snippet above:

Task Description
music:alias Creates a Bottles alias for the application directory
music:chrome Runs the application in Katana and opens the default page in Chrome in _watched_ mode
music:firefox Runs the application in Katana and opens the default page in Firefox in _watched_ mode
service:start Attempts to start the named service if it is installed
music:restart Touches the web.config file of the application to force the application to recycle if running under IIS/IISExpress
music:run Runs the application in Katana and opens the default url in the default browser
