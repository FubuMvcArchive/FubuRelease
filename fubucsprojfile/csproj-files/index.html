CsProj Files

FubuCsProjFile's functionality for manipulating *.csproj files is largely taken from the MonoDevelop codebase.

The key class for manipulating *.csproj files is unsurprisingly called CsProjFile.

            var project = CsProjFile.LoadFrom("FubuMVC.SlickGrid.Docs.csproj");

            // Add new references, code items, etc.

            project.Save();

As of now, the CsProjFile class allows you to create, load, and save *.csproj files while editing basic properties and managing code items and assembly references.
