Rake Files

The rake transform can either create a brand new rakefile in the root folder of a code repository or add new content to an existing rakefile or rakefile.rb file in the root directory.

To add content to a rake build script, add a file named rake.txt to the template directory.

The rake.txt just contains templated Ruby script to add to the code's build script. As an example, this is the rake.txt file from the basic "fubu new" solution:

require 'fuburake'

@solution = FubuRake::Solution.new do |sln|
    sln.compile = {
        :solutionfile => '%SOLUTION_PATH%'
    }

    sln.assembly_info = {
        :product_name => "%SOLUTION_NAME%",
        :copyright => 'Copyright 2013. All rights reserved.'
    }

    sln.ripple_enabled = true
    sln.fubudocs_enabled = true
end

The rake transform applies the substitutions from the template request and plan. The rake transform attempts to be idempotent such that it will not duplicate text in the existing rakefile.
