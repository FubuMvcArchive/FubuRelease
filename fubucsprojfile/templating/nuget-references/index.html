Nuget References

References to nuget dependencies are added via ripple using Ripple's batch installation support. If you're using Ripple for nuget management, a call to ripple restore will pull down any new nugets from the templating.

To declare nuget dependencies in project templates, add a file called nuget.txt to the template folder. List nuget dependencies by name on each line. If you need to specify a specific nuget version, use a slash then the version number after the name.

Here is an example of a nuget.txt file from the FubuMVC.RavenDb template in fubu.exe:

StructureMap/2.6.3
FubuMVC.RavenDb/0.9.5.85
The version number is optional. If no version number is given, Ripple will use the latest version that it can find.
