AssemblyInfo Alterations

To add assembly level attributes to a project's AssemblyInfo.cs file, place a file named assembly-info.txt in a template folder.

The main use case for this transform was to add the [FubuModule] attribute in fubu.exe's Bottles template in this file.

using FubuMVC.Core;

[assembly: FubuModule]
