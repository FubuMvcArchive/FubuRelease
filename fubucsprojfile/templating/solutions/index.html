Solution Templates

Templates for new solutions are kept in the /solutions folder in your template library.

Solution templates can contain:

  1. Instructions
  2. Rakefile transforms
  3. Gems requirements
  4. Nuget requirements
  5. Files and folders
  6. Gitignore entries

Everything in a solution template will be applied to the root of the code repository.
