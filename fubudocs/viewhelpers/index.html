Topic View Helpers

Link to a Topic in the same Project

To link to a topic within the same project, just use:

<TopicLink name="codesnippets" />

which renders:

Code Snippets

Link to a Topic in a different Project

To link to a topic in a different project, use: More about Code Snippets

Note that in the second usage you'll need to specify the project name as part of the name.

Link to another Project

<ProjectLink name="FubuMVC.CodeSnippets" />

LINK TO PROJECT 'FubuMVC.CodeSnippets'

Table of Contents

You can add a table of contents for all the topics beneath the current topic by using the <TableOfContents /> binding.

Project Summary

You can display the project summary for the current project with the <ProjectSummary /> binding.

Information Balloons

As a convenience method, use the <Info>[text]</Info> binding as a shortcut for Twitter Bootstrap balloons like this:

I'm an informational bullet!
