Enhancing Existing Projects

fubu enhanceEnhances an existing project by applying infrastructure templates to an existing project

The fubu enhance command allows you to add named infrastructure features to existing projects. 
fubu enhance MyProject option1 option2
Today, the only usable options are _serenity_ or _storyteller_ to add Serenity or Storyteller2 testing setup to an existing project.
This command will get many more options for validation setup, authentication, and other options in later versions.
fubu enhance [] [-l, --list]
Arguments
projectName of the project to enhance
optionsThe options to apply
Flags
[-l, --list]List all the valid options
