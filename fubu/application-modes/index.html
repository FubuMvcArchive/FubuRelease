Application Modes

fubu modeReads and sets the 'FubuMode' environment variable

A handful of things in FubuMVC behave differently if the application is in _Development_ mode. The fubu mode command gives you a quick way to change the FubuMode on your application box. To set your machine to development mode, use: 
fubu mode --dev
To clear the FubuMode to nothing, use: 
fubu mode --clear
To set the FubuMode to something different, use: 
fubu mode --mode [your custom mode]

fubu mode[-m, --mode ] [-d, --dev] [-c, --clear]
Flags
[-m, --mode <mode>]Named mode for modes other than Development
[-d, --dev]Sets the environment variable to FubuMode=Development
[-c, --clear]Clears the 'FubuMode' environment variable
