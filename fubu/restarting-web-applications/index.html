Restarting Web Applications

fubu restartRestarts a web application by 'touching' the web.config file

If you are running your application with IIS, fubu run, or any other server that would automatically refresh your application if the web.config file changes, you can use the fubu restart command to touch your configuration file to force an application restart on the local box. The usage is: 
fubu restart [path to the application]
