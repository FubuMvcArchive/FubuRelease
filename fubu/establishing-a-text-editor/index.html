Establishing a Text Editor

fubu editorChoose or display the path to the editor that you wish fubu.exe to use when opening files

Mostly for FubuDocs, but possibly for other usages later, the fubu editor command stores an environment variable to your prefered text editor that will be used to open text files from fubu related tools.
fubu editor
Arguments
pathPath to the replacement editor
